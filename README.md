# vue-cnode
[预览链接](https://nestlecafe.github.io/vue-cnode-website/)


*vue 2.0 + Axios + Vue-router实现CNode社区*

这是我在学习Vue时练手的小作品，使用cnode社区提供的api实现

实现了话题列表、话题内容和回复、用户个人信息页面，做了少许响应式。

还有许多待改进的地方
